## Tomokichi Globe & Story

3D 地球儀上で `By Region` と `By Trip` を切り替えながら、ともきちの旅行記録を辿る Next.js アプリです。

## Getting Started

開発サーバー起動:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いて確認します。
