# AGENTS.md

## Project Context
**Project Name:** Tomokichi Globe & Story
**Goal:** Create a 3D Earth visualization travel log with "Region" and "Trip" (Scrollytelling) modes.
**Tech Stack:** Next.js (App Router), TypeScript, Tailwind CSS, Mapbox GL JS (via react-map-gl), pnpm.

## Key Agents & Roles
- **Antigravity (You):** Lead Developer. Responsible for overall architecture, Mapbox integration, and Scrollytelling logic.

## Conventions
- **Package Manager:** `pnpm` (Start commands with `pnpm`).
- **Styling:** Tailwind CSS.
- **Map Library:** `react-map-gl` + `mapbox-gl`.
- **State Management:** React Context or simple State for MVP.
- **Testing:** Vitest (Unit), Playwright (E2E).

## Important Commands
- `pnpm dev`: Start dev server.
- `pnpm test`: Run unit tests.
- `pnpm test:e2e`: Run E2E tests.
